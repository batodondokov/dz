# Экзаменнационное задание

Объект исследования - https://lpsn.dsmz.de/genus/Acinetobacter Acinetobacter

Шаг 1. Были скачаны 20 беловых последовательностей разных штаммов Acinetobacter baumannii, с помощью ncbi-datasets-cli

Шаг 2. Через python скрипт выбрал уникальные последовательности, функция которых не определена (hypothetical protein, HP) и сохранил их в файл (acinetobacter_20_HPs.faa).

Шаг 3. Передал белки HP в DeepSecE на интерфонике. Получил таблицу с результатами классификации (results.csv).

Шаг 4. На ВМ получил результаты kofam и pfam. Через python скрипт определил по kofam и pfam наличие генов T6SS и сохранил в таблицу (t6ss_presence.csv).

Шаг 5. Скачал gbff файлы, чтобы достать 16S rRNA. Записал 16S rRNA каждого штамма в один fasta файл. В программе MEGA выровнял гены и построил дерево, экспортировал его в формате nwk. С помощью скрипта отобразил дерево и добавил информацию о налиичии T6SS (result.png).
